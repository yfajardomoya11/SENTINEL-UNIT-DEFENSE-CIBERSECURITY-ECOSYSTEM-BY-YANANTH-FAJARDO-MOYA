# 🛡️ Cybersecurity Ecosystem – Yananth Fajardo

A modular cybersecurity platform simulating real-world Security Operations Center (SOC) operations.

---

## 🔗 Components

### 🌐 Network Security Tool

Manages network devices, applies configurations and performs secure backups.

🔗 https://github.com/yfajardomoya11/Network-Security-Tool

---

### 🚨 Sentinel Watch

Intrusion Detection System (IDS) for monitoring network traffic and detecting threats.

🔗 https://github.com/yfajardomoya11/Sentinel-Watch

---

### 🛡️ Guardian Engine (Sentinel Shield)

Event-driven security engine for log analysis and threat monitoring.

🔗 https://github.com/yfajardomoya11/Guardian-Engine

---

## ⚙️ How It Works

1. Network Security Tool manages infrastructure
2. Sentinel Watch detects suspicious activity
3. Guardian Engine analyzes logs and events

---

## 🎯 Objective

Simulate a real-world Blue Team / SOC environment using modular cybersecurity tools.
